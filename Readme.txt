The above code is written in cypress framework

Please use following command to install the cypress 
1 - npm install cypress --save-dev

2- Test case file is under folder e2e >>Fancode_spec.js

3- To open or run the test file either run    command npx cypress open and select the file to run the test 

Alternatively  you can also run via command line as from root directory by running command as 

npx cypress run --spec= "cypress/e2e/fancode_spec.js"
